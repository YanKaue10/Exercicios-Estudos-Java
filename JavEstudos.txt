Expressão for(); 

Estrutura planejada para ter uma quantidade determinada de expressões. Onde a primeira você inicializa a variável, a segunda é a expressão, onde vc determina se vai voltar TRUE or FALSE, e a ultima a manipulação da variável de controle para que esse for para de executar. 
 
 
Nessa estrutura segue a mesma lógica do while, mas um pouco mais certeiro na quantidade que será determinada. 

Ex: 

for(int contador = 0; contador <=20; contador += 2);Enquanto o contador for menor ou igual a 20, ele vai adicionar de 2 em 2;

Expressão DO/WHILE();
 
Nessa expressão, no do{} vc tem um bloco de código onde não tem nenhuma expressão associada ao do, mas sim ao while, que está depois do do. E essa é a única estrutura de controle que termina com ;. 

Mas porque ela é a unica expressão que termina com ";"? 

O do/while é a única estrutura de repetição em Java que exige ponto e vírgula (;) após a condição, porque ela é a única em que a condição vem depois do bloco de código.


Ex da estrutura:

do{
  //bloco de codigo
} while(condição);

Classe vc Objetos -

Classe - 
Uma classe define um tipo de objeto, ou seja, representa a estrutura de dados: atributos, métodos, construtores...

Exemplo de classe aluno: temos atributos nome, idade e curso. Além de métodos assistirAula(), fazerProva() e calcularNota(); 
 
Os métodos são criados definindo seu tipo de retorno, no caso o tipo "void", que retorna vazio. 


Objetos - 

Um objeto é ums instancia criada a partir de uma determinada classe.

A classe AlunoTeste representa o objeto, dentro dela é preciso chamar o método main. 

Após instanciar o objeto é possível declarar valores aos atributos e chamar os métodos. 


Atributos - 

Os atributos são as características de um objeto, também podemos chamar de variáveis. 

Exemplos de atributos: nome, idade, peso, altura...


Construtores - 
São métodos responsáveis pro criar os objetos instanciando a classe que foi definida; 





Dentro do corpo da classe nos temos os atributos e comportamentos dessa classe. 

Os atributos são dados e os comportamentos são métodos

Por exemplo:
Dados = int, boolean, string, date...

Os métodos são os comportamentos dessa classe.

Atributos e comportamentos são chamados de MEMBROS de uma classe. 

Instância é sinônimo de objeto, ou seja, uma classe tem objeto A, objeto B e objeto C, pode ser também chamado de instancias da classe.

Quando você coloca static em um atributo (ou método) de uma classe em Java, está dizendo:

"Esse atributo não pertence a um objeto específico, e sim à própria classe."
 
Ou seja, todos os objetos dessa classe compartilham o mesmo valor desse atributo.
 


This e this(). 

Em Java, a palavra-chave "this" é usada para se referir ao objeto atual da classe. Ela é usada para diferenciar variáveis de instância de variáveis locais com o mesmo nome, para invocar métodos da própria classe e para chamar construtores dentro de outros construtores da mesma classe. 

Ex: 

Acessando atributos:

class Pessoa {
    String nome;

    public Pessoa(String nome) {
        this.nome = nome; 
// 'this.nome' refere-se ao atributo da classe, 'nome' refere-se ao parâmetro do método
    }
} 


Chamando construtores:

class Carro {
    String marca;
    String modelo;

    public Carro() {
        this("Desconhecida", "Desconhecido"); // Chama o construtor com dois argumentos
    }

    public Carro(String marca, String modelo) {
        this.marca = marca;
        this.modelo = modelo;
    }
}
